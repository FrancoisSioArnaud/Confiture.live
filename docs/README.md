# Documentation — Confiture.live

Ce dossier regroupe les specs produit, visuelles et techniques prêtes à être utilisées pour lancer le développement avec Codex.

## Fichiers

- `product-spec.md` : spec fonctionnelle complète du produit de gestion de jam.
- `visual-spec-table-cards.md` : spec UX/UI du tableau, des cartes, des drawers et des états visuels.
- `technical-stack.md` : stack technique recommandée.
- `codex-ready-technical-spec.md` : décisions techniques verrouillées avant développement.
- `../AGENTS.md` : règles de travail exhaustives pour cadrer Codex.

## Source de vérité
